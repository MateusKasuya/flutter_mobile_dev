## Decisões tomadas

- Usar o pacote `http` (mais simples que `dio` para iniciantes)
- Função `login()` como top-level function em vez de classe/service
- `LoginScreen` aceita `loginFn` como parâmetro para facilitar testes (injeção de dependência simples)
- URL construída com `Uri.http(host, path)` para separar responsabilidades de host e rota
