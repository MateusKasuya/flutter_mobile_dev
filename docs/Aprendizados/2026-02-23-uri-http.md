# Construindo URLs com Uri.http() em Dart

[[Aprendizados/_index|Aprendizados]]

---

## O que aprendi

Em Dart existem duas formas principais de criar uma URL para requisições HTTP:

- `Uri.parse()` — recebe a URL completa como string
- `Uri.http()` — separa o host do path (caminho)

## Por que importa

`Uri.http()` é mais organizado: o host fica numa constante separada e o path é passado como argumento. Isso facilita reutilizar o host para múltiplos endpoints.

## Exemplo prático

```dart
// com Uri.parse — URL toda junta
final url = Uri.parse('http://fretefacilweb.ccmcloud.com.br:8624/sftlogin/login');

// com Uri.http — host separado do path
const String _baseUrl = 'fretefacilweb.ccmcloud.com.br:8624';
final url = Uri.http(_baseUrl, '/sftlogin/login');
```

Ambos produzem a mesma URL. A segunda forma escala melhor quando o app tem vários endpoints, pois o host fica em um único lugar.
